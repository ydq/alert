jQuery/Zepto 页面弹窗插件

下载 
<script type="text/javascript" src="jquery.js"></script>
<script type="text/javascript" src="alert.js"></script>
<script>
	$.alert('弹出了一个alert对话框')
	$.confirm('弹出了一个confirm对话框',function(e){
		$.alert('你点击了'+(e?'确定':'取消'))
	})
	$.tips('显示了一条tips')
	$.load('显示了一条load信息')
</script>
方法 参数 必 参数说明 方法说明
$.alert(msg[,fn]) string 是 弹窗消息 弹出一个只有确定按钮的对话框
function 否 回调函数
$.confirm(msg[,fn]) string 是 弹窗消息 弹出一个有确定和取消两个按钮的对话框
function 否 回调函数
$.tips(msg[,time]) string 是 消息内容 弹出一个会自动消失的tips提示框
int 否 自动关闭的时间，仅PC浏览器有效
$.load(msg) string 是 消息内容 弹出一个没有任何操作按钮的对话框，需要手动关闭
$.loaded() - - - 关闭load对话框
